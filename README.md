# Distributed Payment Network

A distributed system demonstrating master-agent architecture for payment synchronization across multiple nodes.

## Architecture

- **Master Service**: Central coordination and data aggregation
- **Agent Service**: Autonomous payment processing nodes
- **Common**: Shared DTOs and utilities

## Tech Stack

- Backend: Spring Boot 3.x
- Frontend: React + TypeScript
- Database: H2 (development)
- Sync: REST + Scheduled tasks
